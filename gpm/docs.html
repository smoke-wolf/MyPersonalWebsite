Installing GHPM

Upon completion of the download, the next step is to extract the contents of the file.

Inside, you will find a directory denoted as "GitHub-Package-Manager." However, it's essential to designate your preferred storage location for this folder before advancing.

Subsequently, in your command line interface, direct yourself to the "GitHub-Package-Manager" directory by employing the 'cd' command.

To initialize the application, execute the 'python3 Start.py' command.

With these actions completed, you are now well-prepared to fully leverage the capabilities of GitHub-Package-Manager.

Installing Repositories with GHPM

Load GHPM:

Launch the GHPM application.

Initiate Installation: Within the GUI, select the "Install" option. This selection will prompt a new window.

Specify Repository or Folder: Now, you can automate the installation process between GitHub and local folders. v Enter the Git repository you wish to download, or choose a local folder for connection.

Submission: Click the "Submit" button, and you will be presented with a file display.

Automated Installation: The displayed window allows for complex automated installations. After selecting a file, the window will close, and the system will automatically manage the required environment for launches, completing the installation.

Activating Downloads

Repo Selection: If you've downloaded a repository from the Installer, it will appear within the "Activator" in the GUI. Standard installations are found in the first layer of the "Activator," which will request you to confirm GitHub.

Confirmation: Once you select "Yes" for GitHub, you'll see the following display. If you choose "No," it will prompt for local applications, and another "No" will display complex applications.

Application Launch: Select and double-click the option for your application. This action opens a new window reflecting the application as it launches in the chosen format.

Please note that the specific steps for activation may vary depending on the style of the package and the installation method

Uninstall Downloads

Access Uninstall: From the main menu in the GUI, select the "uninstall" option.

Uninstall Menu: This will open a menu that allows you to uninstall both GitHub and Advanced applications.

Confirmation: Once you've made your selection, click "Confirm Uninstall." This action will bring up a confirmation window.

Uninstall Process: If you select "Yes," the program will attempt to uninstall the application. Depending on how it manages the uninstallation, you will receive a corresponding notification.
